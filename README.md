💼 Portfólio — Carrinho de Compras (Estilo Shopee)
Simulação de carrinho de compras inspirado em plataformas de e-commerce.
A aplicação permite gerenciar produtos dinamicamente, incluindo nome, preço unitário, quantidade e cálculo automático do valor total.
🎯 Objetivo
Desenvolver um sistema de carrinho de compras funcional que simule operações reais de um e-commerce:
- Adicionar produtos
- Remover itens
- Calcular subtotal e total
- Implementar lista de desejos (wishlist)
🧠 Processo Criativo
O desenvolvimento foi baseado na separação de responsabilidades:
- Item: dados e cálculo individual de cada produto
- Cart: gerenciamento do carrinho e regras de negócio
- App: execução e testes das funcionalidades
Essa abordagem facilita manutenção, reutilização e expansão do sistema.
⚙️ Funcionalidades
- Adicionar produtos ao carrinho
- Remover produtos por nome ou índice
- Calcular subtotal dos itens
- Calcular valor total
- Exibir carrinho no console
- Gerenciar lista de desejos (wishlist)
🧱 Arquitetura
- Estrutura modular com separação por arquivos
- Uso de classes para organização da lógica
- Código escalável e de fácil entendimento
🛠️ Tecnologias Utilizadas
- JavaScript (ES6+)
- Programação Orientada a Objetos (POO)
- Módulos JS
📈 Conclusão
Este projeto demonstra a aplicação prática de conceitos fundamentais de programação, com foco em organização, reutilização de código e construção de lógica escalável, servindo como base para sistemas mais complexos de e-commerce.
